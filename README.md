# 学习 Telegram

Telegram 太顺滑了，动画丰富，UI精美。有必要进一步学习。

但是，刚将项目跑起来，一个 18k 行的 ChatActivity 映入眼帘，直接震撼了我幼小的心灵。

这不得不成为一个长期计划。可能中途会断一段时间。因此需要 git 帮我保存进度。
